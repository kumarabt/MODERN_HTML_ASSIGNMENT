DIFFERENCE BETWEEN HTML AND HTML5

HTML HTML5
1. No semantic tags were present in HTML. We can only use div tag as a container
for our diffrent content sections. 		1. HTML5 introduced various semantic tags that provide meaning and structure to our content.
for eg: section tag, nav tag, main tag, header and footer tag etc.
2.There was no access to advanced graphics tags in HTML.
Only jpeg and png formats can be used which becomes ruptures when image
height and width are increased		 2. HTML introduced users to new advanced graphics tags like SVG and CANVAS
that resolved the problem of image rupturing at large height and width.
3. Previously HTML doesn't have inbuilt audio and video support in it.
So, we have to use third party plugins like adobe flash for audio and video support. 		3. HTML5 on the other hand has inbuilt audio and video support in it.
That, can be accessed by using audio and video tags in HTML5.
