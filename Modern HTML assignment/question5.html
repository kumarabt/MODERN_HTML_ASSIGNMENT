3 KEY FEATURES OF HTML5

1.SEMANTIC TAGS:

HTML5 introduced a new set of semantic tags that is used to create meaningful webpages. Since, previously only div tag can be used as a container as we define class in that as header, footer, and nav etc. in it. But, now after HTML5 we can directly used tags like <header>,<footer>,<article>,<nav> and <section> etc. and can add meaning to our webpages. this provides better clarity of content type and helps in organization of content, that makes it easier for search engines and web crawlers to understand and interpret web page structure.

2. AUDIO AND VIDEO SUPPORT:

HTML5 includes inbuilt support for audio and video elements through the <audio> and <video> tags. This eliminates the need for third party plugins like Adobe Flash, that's need to be done previously. And allow developers to embed audio and video content directly into the web pages.

3.HTML FORM ADVANCEMENTS:

HTML5 introduced several form input types and attributes that improves user experience and validation. New input types include 'email','url','number' 'date' and 'range' etc. which help browsers to provide better input validation. Additionally, HTML5 also introduced 'placeholder' attribute in input fields and the required 'attribute' for mandatory form fields, and the <datalist>> element for providing predefined options for input fields.
